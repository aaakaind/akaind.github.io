üîê Employee & Admin Login - Enhanced Security

Our employee portal features military-grade 2-stage security checkpoint before login to protect sensitive company data and employee information.

üõ°Ô∏è 2-Stage Security Checkpoint Process

STAGE 1
üîç Pre-Authentication Verification
  • IP Address Verification: Your IP is checked against our whitelist and geolocation database
  • Device Fingerprinting: Browser and device characteristics are analyzed
  • Network Security Check: VPN detection and proxy verification
  • CAPTCHA Challenge: Verify you're human (not a bot)
  • Security Question: Answer a pre-configured security question

‚è±Ô∏è Estimated time: 30-60 seconds

STAGE 2
üîê Multi-Factor Authentication
  • Primary Authentication: Company email + strong password
  • MFA Code: 6-digit code via authenticator app (Google/Microsoft Authenticator)
  • SMS Backup: Optional SMS verification code
  • Biometric Option: Fingerprint or Face ID (if supported)
  • Hardware Key: YubiKey or similar FIDO2 device (optional)

‚è±Ô∏è Estimated time: 20-30 seconds

üîí Employee Portal Access (High Security)

Security Level: HIGH SOC 2 Type II Compliant
  1. Stage 1: Pre-Auth Checkpoint
    • Navigate to staff.akaind.ca
    • Complete CAPTCHA verification
    • Answer security question
    • Device fingerprint captured automatically
  2. Stage 2: Authentication
    • Enter company email address (@akaind.ca)
    • Enter password (min 12 characters, complexity required)
    • Enter 6-digit MFA code from authenticator app
    • Optional: Scan fingerprint or Face ID
  3. Access granted to employee dashboard
