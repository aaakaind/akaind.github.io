üåê DNS Configuration Guide

To enable subdomain access for all AKA Industries platforms, configure your DNS settings as follows:

Primary Domain

akaind.ca ‚Üí A Record ‚Üí Your Server IP www.akaind.ca ‚Üí CNAME ‚Üí akaind.ca

Wildcard Subdomain (Recommended)

*.akaind.ca ‚Üí CNAME ‚Üí akaind.ca

This enables all subdomains (console, staff, admin, etc.) automatically

Individual Subdomains (Alternative)

console.akaind.ca ‚Üí CNAME ‚Üí akaind.ca staff.akaind.ca ‚Üí CNAME ‚Üí akaind.ca admin.akaind.ca ‚Üí CNAME ‚Üí akaind.ca developers.akaind.ca ‚Üí CNAME ‚Üí akaind.ca partners.akaind.ca ‚Üí CNAME ‚Üí akaind.ca status.akaind.ca ‚Üí CNAME ‚Üí akaind.ca docs.akaind.ca ‚Üí CNAME ‚Üí akaind.ca games.akaind.ca ‚Üí CNAME ‚Üí akaind.ca

SSL/TLS Certificates

Enable wildcard SSL certificate for *.akaind.ca to secure all subdomains

Certificate: *.akaind.ca + akaind.ca
